# ヘブライ語Alefbet道場

古典ヘブライ語の子音・母音記号を択一クイズで学習できるWebアプリ。
旧約聖書を原典で読むための第一歩として制作。

**本番URL：https://alefbet.jp**

---

## 機能

**ヘブライ文字クイズ（28問）**
- ヘブライ文字の字形を見て名前を答える4択クイズ
- ランダム順 / Alefbet順を選択可能

**母音記号クイズ 記号→音（16問）**
- 母音記号（ニクド）を見て音を答える6択クイズ（a / i / u / e / o / 無音）
- シェワとカメツは同じ字形で音が異なるため、2つ選んで正解の2択問題
- ランダム順 / 固定順を選択可能

**母音記号クイズ 記号→名前（16問）**
- 母音記号（ニクド）を見て名前を答える4択クイズ
- ランダム順 / 固定順を選択可能

**共通機能**
- タイムアタック＆正答率（%）表示
- フォント切り替え（Cardo / Frank Ruhl Libre / Heebo / Assistant）
- 選択したクイズ種別・出題モード・フォントをlocalStorageで保持
- X（旧Twitter）への正答率・タイムシェア・LINEへのアプリ紹介シェア

## 実装の工夫

**ヘブライ語Unicodeの扱い**
ヘブライ文字は右から左（RTL）に記述され、母音記号（ニクド）は基底文字にUnicode結合文字を重ねて表現する。フォントによってニクドの描画サイズが異なるため、フォントごとにプレビューサイズを個別調整している。

**2択問題のロジック**
有音シェワ（`בְ`）と大カメツ / 小カメツ（`בָ`）は字形が同一で読みが異なる。これをクイズとして出題する際、1問に対して2つの選択肢を両方選ぶと正解となる独自のインタラクションを実装。選択順は問わず、誤答を選んだ時点で即終了する。

**結果表示**
クイズ終了後に正答率（%）とタイムを表示。シンプルな指標で自分の習熟度とスピードを把握できる。
